# SEO do site

Esta fase adiciona:

- seo-site.js
- robots.txt
- sitemap.xml
- site.webmanifest

O `seo-site.js` cria metas para título, descrição, Open Graph, Twitter Card, canonical e dados estruturados JSON-LD.

Também melhora a página `produto.html` com título, descrição e dados do produto para compartilhamento.

Observação: como o site é estático, SEO avançado por produto pode exigir geração estática ou backend futuramente.
