# Áudio do player xamânico

Envie a música principal do site para esta pasta com o nome exato:

```txt
xamanico-ambiente.mp3
```

Caminho público esperado pelo player:

```txt
assets/audio/xamanico-ambiente.mp3
```

O player também aceita faixas opcionais:

```txt
tambor-floresta.mp3
cristais-incensos.mp3
```

Observação: o upload pelo Admin salva a música localmente no navegador usando IndexedDB. Para a mesma música tocar para todos os visitantes do site, o arquivo MP3 precisa ser enviado para esta pasta no GitHub.
