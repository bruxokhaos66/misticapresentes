## Regra daqui para frente

- Novas auditorias devem entrar nesta pasta.
- A raiz do repositorio deve ficar reservada para README, configuracoes essenciais e arquivos de entrada do projeto.
- O status operacional vigente deve permanecer em `STATUS_ATUAL.md`.
