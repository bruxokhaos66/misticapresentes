# Mística Presentes V2

Nova versão do site criada em uma pasta isolada para reconstruir o visual com calma, sem apagar a versão atual.

## Objetivo

Criar um frontend mais limpo, moderno, comercial, sofisticado e xamânico para a Mística Presentes.

## Regras da V2

- Manter a Isis e as imagens já usadas no projeto.
- Manter as funções principais: produtos, carrinho, Pix, WhatsApp, estoque e painel interno.
- Evitar arquivos de correção sobrepostos.
- Separar visual e lógica em arquivos claros.
- Fazer alterações por etapas pequenas.

## Estrutura inicial

- `index.html`: página inicial da V2.
- `v2.css`: visual xamânico/comercial da V2.

## Observação

Nesta primeira etapa, a V2 ainda reaproveita o `app.js` da raiz para preservar carrinho, Pix, WhatsApp, produtos e estoque sem reescrever a lógica funcional.
