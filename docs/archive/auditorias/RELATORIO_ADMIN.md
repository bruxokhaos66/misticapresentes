# Relatório do Admin

Esta fase adiciona um relatório visual ao Admin da Mística Presentes.

## Recursos

- Total de pedidos.
- Pedidos pendentes.
- Pedidos com pagamento confirmado.
- Pedidos em separação ou prontos para retirada.
- Pedidos com estoque baixado.
- Pedidos criados pela Isis.
- Faturamento sem pedidos cancelados.
- Quantidade de itens com estoque baixo.
- Ranking de produtos mais vendidos.
- Lista de itens com estoque baixo.

## Arquivos

- `admin-report.js`
- `product-extras.js`
- `commercial.css`

## Como testar

1. Abrir o Admin.
2. Conferir a seção `Relatórios`.
3. Clicar em `Atualizar relatório`.
4. Validar pedidos, estoque baixo e produtos mais vendidos.
