# Painel de pedidos da Isis

Esta fase melhora o painel de pedidos do Admin.

Recursos adicionados:

- Identificação de origem do pedido.
- Filtro por Todos, Isis, Site/API e Manual.
- Contador de pedidos por origem.
- Contador de pedidos pendentes.
- Destaque visual para pedidos criados pela Isis.
- Mensagem de WhatsApp incluindo a origem do pedido.

Arquivos alterados:

- pedido-status.js
- commercial.css

Como testar:

1. Criar um pedido usando a Isis.
2. Abrir o Admin.
3. Conferir o painel de pedidos.
4. Usar o filtro `Isis`.
5. Confirmar Pix ou alterar status normalmente.
