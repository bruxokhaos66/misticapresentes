# Isis Kits

Esta fase melhora a Isis para montar kits automáticos com produtos reais da API.

Kits adicionados:

- Kit Proteção
- Kit Limpeza Energética
- Kit Presente Místico
- Kit Relaxamento
- Kit Prosperidade
- Kit Amor e Harmonia

A Isis procura produtos disponíveis em estoque, separa por função e permite adicionar o kit ao carrinho.

Arquivos alterados:

- isis-commerce.js
- commercial.css

Como testar:

1. Abrir a área Isis.
2. Perguntar por `kit proteção` ou `kit limpeza energética`.
3. Verificar se aparece um kit com itens reais.
4. Clicar em `Adicionar kit ao carrinho`.
