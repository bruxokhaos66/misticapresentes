# Isis envio de kit

Esta fase adiciona envio do kit sugerido pela Isis para o WhatsApp.

Arquivos:

- isis-kit-share.js
- product-extras.js

Funcionamento:

1. A Isis monta um kit automático.
2. O site adiciona o botão `Enviar kit no WhatsApp`.
3. O botão abre uma mensagem pronta com itens e total sugerido.

Como testar:

1. Abrir a área Isis.
2. Pedir `kit proteção`.
3. Clicar em `Enviar kit no WhatsApp`.
