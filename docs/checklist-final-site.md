# Checklist final do site — Mística Presentes

Use este checklist depois das etapas visuais para validar o site antes de novas melhorias grandes.

## 1. Primeira dobra / Hero

- [ ] Logo aparece corretamente.
- [ ] Menu abre e navega sem esconder conteúdo.
- [ ] Isis não pula de posição após atualizar a página.
- [ ] Isis não fica colada no topo.
- [ ] Texto principal fica legível em desktop.
- [ ] Texto principal fica legível em celular.
- [ ] Botões “Ver produtos” e “Chamar no WhatsApp” funcionam.
- [ ] Cards “Retirada local”, “Pedido rápido” e “Curadoria” ficam proporcionais.

## 2. Experiência sonora

- [ ] Botão aparece bem posicionado.
- [ ] Música não inicia sozinha.
- [ ] Música inicia apenas após clique do cliente.
- [ ] Controle de volume funciona.
- [ ] Botão de desligar funciona.
- [ ] Não atrapalha a leitura no celular.

## 3. Comprar por intenção / Categorias

- [ ] Cards ficam alinhados em desktop.
- [ ] Cards ficam em uma coluna ou duas colunas corretas no celular/tablet.
- [ ] Clique leva para produtos ou aciona sugestão corretamente.
- [ ] Textos não ficam cortados.

## 4. Produtos

- [ ] Cards de produtos têm altura proporcional.
- [ ] Preços aparecem corretamente.
- [ ] Botões de quantidade funcionam.
- [ ] Produto adicionado aparece no carrinho.
- [ ] Layout não fica amontoado.
- [ ] No celular, produtos ficam legíveis.

## 5. Carrinho / Checkout / Pix

- [ ] Carrinho mostra produtos adicionados.
- [ ] Total calcula corretamente.
- [ ] Dados do cliente são preenchidos corretamente.
- [ ] QR Code Pix aparece quando esperado.
- [ ] Envio para WhatsApp funciona.
- [ ] Mensagem enviada contém produtos, total e dados necessários.

## 6. Isis assistente

- [ ] Campo de conversa aparece corretamente.
- [ ] Sugestões da Isis funcionam.
- [ ] Imagem da Isis na seção inferior está correta.
- [ ] A seção não fica pesada ou cortada no celular.

## 7. Admin / Painel

- [ ] Painel admin permanece escondido sem acesso correto.
- [ ] Login/admin não aparece para cliente comum.
- [ ] Atividade recente aparece corretamente.
- [ ] Botões de WhatsApp no admin funcionam.
- [ ] Contador de acessos aparece no ADM quando disponível.
- [ ] Fallback local do contador não quebra a página.

## 8. Rodapé

- [ ] WhatsApp está correto.
- [ ] Instagram está correto.
- [ ] Domínio está correto.
- [ ] Visual combina com o topo.
- [ ] Rodapé fica legível no celular.

## 9. Performance e cache

- [ ] Abrir em aba anônima e comparar com navegador normal.
- [ ] Usar Ctrl + F5 após deploy.
- [ ] Conferir se nenhum script aparece com erro no console.
- [ ] Verificar imagens carregando sem quebrar.
- [ ] Verificar se o site não trava durante rolagem.

## 10. Testes por dispositivo

- [ ] Desktop largo.
- [ ] Notebook.
- [ ] Celular Android.
- [ ] Navegador Chrome.
- [ ] Navegador Edge/Opera.

## Observação

Novas melhorias visuais devem ser feitas em PR separado e com validação por print/vídeo antes de mesclar.
