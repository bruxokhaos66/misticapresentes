# Fase 2 - Catálogo moderno

Esta fase melhora a experiência de compra no site da Mística Presentes sem refazer todo o visual.

## Melhorias aplicadas

- Busca por produto, categoria, descrição e selo comercial.
- Filtro por categoria gerado automaticamente pelos produtos cadastrados.
- Ordenação por destaques, menor preço, maior preço, mais estoque e nome A-Z.
- Selos comerciais nos cards:
  - Mais vendido
  - Promoção
  - Novidade
  - Poucas unidades
  - Sob encomenda
  - Loja parceira
  - Kit especial
- Campo de selo comercial no cadastro de produto do painel Admin.
- Estado vazio quando nenhuma busca encontra resultado.
- Layout responsivo para celular.

## Como testar

1. Abrir a vitrine do site.
2. Buscar por nome de produto, categoria ou intenção.
3. Filtrar por categoria.
4. Alterar a ordenação.
5. Cadastrar um produto no Admin com selo comercial.
6. Verificar se o selo aparece no card.
7. Testar no celular.
